                          package
change <placeholder> with an argument

Usage:
	pkg -<optional: method> <author>/<package name>

Types of argument:
	method:
		-p: pip
		-g: git
		none: direct
	author of a package
	package name

Example:
	pkg -g umanochiocciola/platformer
